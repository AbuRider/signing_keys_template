# signing_keys_template
Script Template for generate signing keys build. Original repo: https://github.com/ProjectMatrixx/android_vendor_lineage-priv_keys-template.git

How to use?
git clone --depth=1 https://android.googlesource.com/platform/development

And then clone this repo
git clone https://github.com/AbuRider/signing_keys_template.git vendor/lineage-priv/keys

./generate.sh
